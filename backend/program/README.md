# Anchor program (Solana)

Place the TicketChain Anchor program here (e.g. `anchor init` or copy an existing program).

**Target instructions (from architecture doc):**

- `create_event` – Event PDA, tiers, organizer
- `buy_ticket` – Mint NFT, transfer SOL
- `list` – Listing PDA, escrow
- `buy_listing` – Split SOL, transfer NFT

Program ID and deployment steps go in this folder once the program is added.
